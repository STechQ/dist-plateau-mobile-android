## Component Styling

All components support:

- **CSS Styling** - Custom CSS classes
- **Theme Integration** - Light/dark themes
- **Style Props** - Inline styling
- **Responsive Design** - Breakpoint-based

### Example

```javascript
{
  "type": "VButton",
  "props": {
    "text": "Click Me",
    "color": "primary",
    "variant": "contained",
    "disabled": false
  },
  "style": {
    "marginTop": 16,
    "borderRadius": 8
  }
}
```
